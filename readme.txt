A modular all-in-one solution for WordPress SEO, performance, security and administration.

== Description ==

SEO Wunderkiste combines 17 powerful features in one plugin. To protect your site's performance, all modules are disabled by default. Enable only the modules you need under "Settings > SEO Wunderkiste".

= SEO & Content (5 Modules) =

* **SEO Meta Settings** - Extended meta tags per page/post with live preview (Title, Description, Open Graph, Twitter Cards)
* **SEO Schema (JSON-LD)** - Add custom Schema.org structured data to posts and pages
* **Bulk NoIndex Manager** - Mass set/remove NoIndex for pages and posts
* **SEO Zombie Killer** - Redirects useless attachment pages to their parent posts (301)
* **Conversion Tracker** - GA4 and Google Ads conversion tracking on individual pages

= Image & Media (5 Modules) =

* **Image Resizer (800px/1200px)** - Scale large images with one click (92% quality)
* **Upload Cleaner** - Automatically clean filenames on upload (umlauts, spaces)
* **Zero-Click Image SEO** - Auto-generate alt-text and titles from filenames
* **Media Inspector** - Show file size and dimensions in media library
* **SVG Upload Support** - Allow SVG uploads with security sanitization

= Performance (1 Module) =

* **Emoji Bloat Remover** - Remove unnecessary WordPress emoji scripts

= Security & Admin (4 Modules) =

* **XML-RPC Blocker** - Disable the XML-RPC interface for brute-force protection
* **Login Guardian** - Hide login page behind a secret parameter
* **Comment Blocker** - Globally disable comments on the website
* **ID Column Display** - Show Post/Page/Media IDs in all overviews (click to copy)

= Content Tools (2 Modules) =

* **Date Shortcode** - Insert current date via shortcode with flexible formats
* **Semantic Blocks** - HTML5 wrapper blocks for better structure and SEO

== Installation ==

1. Upload the 'seo-wunderkiste' folder to '/wp-content/plugins/'
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Settings > SEO Wunderkiste" to enable desired modules

== Frequently Asked Questions ==

= Are all modules active by default? =

No, all modules are disabled by default for better performance. Enable only what you need.

= Is this plugin compatible with other SEO plugins? =

Yes, but we recommend disabling overlapping features to avoid conflicts.

= Does the SVG upload pose a security risk? =

No, SVG files are automatically sanitized on upload. Scripts and event handlers are removed.
